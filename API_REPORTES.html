📊 Sistema de Reportes y Exportación

API de Hospital Monitor - Documentación Completa

📋 Descripción General

El sistema de reportes permite obtener estadísticas detalladas sobre la ocupación de camas, temperatura de refrigeradores y generar reportes exportables en Excel y JSON.

Características Principales:

  • Obtener estadísticas de ocupación de camas por hospital
  • Monitorear temperatura de refrigeradores
  • Generar reportes en Excel con formato profesional
  • Descargar datos en formato JSON
  • Desglose por pisos y áreas
  • Cálculo automático de porcentajes y promedios

📁 Generar Reporte Excel

POST Descargar Reporte

/api/reportes/descargar

Descripción:

Genera un reporte en Excel con los datos solicitados.

Body Request:

Ocupación de Camas:

{
  "hospital_id": 1,
  "tipo_reporte": "ocupacion",
  "fecha_inicio": "2026-06-01",
  "fecha_fin": "2026-06-16"
}

Temperatura:

{
  "hospital_id": 1,
  "tipo_reporte": "temperatura",
  "fecha_inicio": "2026-06-01",
  "fecha_fin": "2026-06-16"
}

Respuesta Exitosa:

200 OK - Archivo Excel

Content-Type: application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
Content-Disposition: attachment; filename="reporte_1_20260617_103000.xlsx"

Características del Excel:

  • Encabezados con estilos profesionales
  • Datos organizados por piso/categoría
  • Cálculos automáticos
  • Ancho de columnas ajustado
  • Bordes y formato aplicados

💻 Ejemplos Prácticos

JavaScript/Fetch:

// Obtener camas
fetch('/api/reportes/camas/1')
  .then(r => r.json())
  .then(data => console.log(data));

// Descargar Excel
fetch('/api/reportes/descargar', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    hospital_id: 1,
    tipo_reporte: 'ocupacion'
  })
})
.then(r => r.blob())
.then(blob => {
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = 'reporte.xlsx';
  a.click();
});

Python/Requests:

import requests

# Obtener estadísticas
response = requests.get('http://localhost:5000/api/reportes/camas/1')
datos = response.json()

# Descargar Excel
payload = {
    'hospital_id': 1,
    'tipo_reporte': 'ocupacion'
}
response = requests.post(
    'http://localhost:5000/api/reportes/descargar',
    json=payload
)
with open('reporte.xlsx', 'wb') as f:
    f.write(response.content)

⚠️ Códigos de Estado HTTP

Código Significado Descripción
200 OK Solicitud exitosa Los datos fueron recuperados correctamente
201 Created Creado El recurso fue creado exitosamente
400 Bad Request Solicitud Inválida Los parámetros enviados son inválidos
500 Internal Server Error Error del Servidor Ocurrió un error al procesar la solicitud
